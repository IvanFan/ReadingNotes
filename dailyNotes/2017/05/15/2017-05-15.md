# Auto Deployment with Docker Image (2)

## Setup QA Deployment and Service

```
# web-deployment.yml
apiVersion: extensions/v1beta1
kind: Deployment
metadata:
  labels:
    name: qa-sl
  name: qa-sl
spec:
  replicas: 2
  template:
    metadata:
      labels:
        name: qa-sl
    spec:
      containers:
      - image: gcr.io/stream-lending/development
        name: qa-sl
        ports:
        - containerPort: 8080
          name: http-server
```

```
# web-service.yml
apiVersion: v1
kind: Service
metadata:
  name: qa-sl
  labels:
    name: qa-sl
spec:
  type: LoadBalancer
  ports:
    - port: 80
      targetPort: 8080
      protocol: TCP
  selector:
    name: qa-sl
```

```
  replicas: 1

    metadata:
      labels:
        name: mongo-qa
    spec:
      containers:
      - image: mongo
        name: mongo-qa
        ports:
        - name: mongo
          containerPort: 27017
          hostPort: 27017
        volumeMounts:
            - name: mongo-persistent-storage
              mountPath: /data/db
      volumes:
        - name: mongo-persistent-storage
          gcePersistentDisk:
            pdName: sl-qa-mongo-disk
            fsType: ext4
```

```
  labels:
```
